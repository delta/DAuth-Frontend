## About

## Setup

- clone the repo
   
   ```
   git clone https://github.com/delta/DAuth-Frontend.git
   ```
- install dependencies
  
    ```
    cd DAuth-Frontend

    npm install
    ```

- Run  `cp env.example.js env.js`
- fill the backend URL in the env.js file
- Run `npm run build`
- Run  `npm run dev`
